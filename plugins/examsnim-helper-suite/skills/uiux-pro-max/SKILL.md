# UI/UX Pro Max

Upgrade user experience like a senior product designer plus frontend engineer.

## Scope

- Information architecture and user flows.
- Layout hierarchy, spacing, rhythm, contrast and typography.
- Responsive behavior across mobile, tablet and desktop.
- Accessibility: semantic HTML, keyboard navigation, labels, focus states, contrast and reduced motion.
- Motion design: meaningful transitions, loading states, microinteractions, scroll effects and page transitions.
- Design-system hygiene: reusable components, tokens, variants, empty/error/success states.

## Rules

1. Do not add heavy animation libraries unless the project already uses them or the benefit is clear.
2. Respect `prefers-reduced-motion`.
3. Keep motion purposeful, fast and non-blocking.
4. Avoid visual changes that break conversion, accessibility, readability or performance.
5. For React/Next.js, prefer small composable components and progressive enhancement.

## Checklist

Review hero clarity, CTA hierarchy, form validation, loading/empty/error states, mobile navigation, touch targets, accessible names, focus rings, heading order, landmarks, images, fonts, animation cost and bundle impact.
