# ExamsNim Codex Helper Plugins

GitHub-ready Codex helper plugin suite for ExamsNim.

This repository contains shortcut-style Codex helpers such as `$superpower`, `$uiux-pro-max`, `$seo-sitemap-pro`, `$test-pro`, `$memory-pro`, `$multi-agent-squad`, `$security-pro`, `$performance-pro`, `$examsnim-pro`, and `$deploy-pro`.

Codex native slash commands such as `/plugins`, `/skills`, `/agent`, `/init`, and `/memories` are built in. The custom helpers in this repo are packaged as Codex skills, so use them with `/skills` or by mentioning the skill name with `$`.

## Install

```bash
codex plugin marketplace add AvinasshSingh/examsnim-platform --ref main --sparse .agents/plugins
```

Then restart Codex, open `/plugins`, choose **ExamsNim Codex Helper Marketplace**, and install **ExamsNim Helper Suite**.

## Use

```text
$superpower build the best ExamsNim feature and test it
$uiux-pro-max audit and upgrade the UI/UX with premium motion
$seo-sitemap-pro check SEO, robots.txt, canonical URLs and sitemap.xml
$test-pro create missing tests and run checks
$memory-pro update durable Codex repo memory
$multi-agent-squad review with SEO, UI/UX, QA, security and performance agents
```

See `docs/codex-slash-helper-prompts.md` for copy-paste prompt aliases like `/superpower` and `/ui/ux pro max`.
